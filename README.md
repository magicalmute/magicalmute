# ModernBlog

A modern, static blog website built with HTML, CSS, and JavaScript. This blog features a clean, responsive design and includes an admin panel for content management.

## Features

- Responsive design
- Blog post management
- Admin panel
- Profile settings
- Site configuration
- Local storage for data persistence

## Setup

1. Clone the repository
2. No installation needed - it's a static website
3. Open index.html in your browser

## Admin Access

Default admin credentials:
- Username: admin
- Password: admin123

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome
- Local Storage for data persistence
